# 📅 ScheduleControl

Sistema de gestión de turnos y citas para negocios pequeños como barberías, peluquerías, consultorios o centros de servicio.

## 🧠 Funcionalidades principales

✅ Gestión de clientes  
✅ Gestión de servicios  
✅ Gestión de negocios  
✅ Gestión de citas  
✅ Editar y eliminar registros  
✅ Prevención de citas cruzadas  
✅ Validación de horario laboral  
✅ Interfaz web moderna  
✅ Navegación intuitiva
